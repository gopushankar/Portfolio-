# Gopu Shankar Portfolio Website

This is a personal portfolio website for Gopu Shankar showcasing his skills, projects, and contact information. The website is designed to be visually appealing, responsive, and user-friendly.

## Features
- Clean and modern design with blue shades and centered content
- Navigation menu for easy access to Home, Photos, About, and Contact sections
- Profile section with image and brief introduction
- Detailed project highlight with description
- About section explaining skills and development approach
- Contact form to reach out directly

## Technologies Used
- HTML5
- CSS3 (custom styles for colors, fonts, margins, and layout)
- JavaScript (for basic form handling and localStorage)

## How to Use
1. Open `index.html` in any modern browser
2. Navigate through different sections using the menu
3. Fill out the contact form to send a message (simulated with localStorage)
